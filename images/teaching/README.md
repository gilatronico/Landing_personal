# Logos de Enseñanza

Esta carpeta contiene los logos de las instituciones donde se imparte docencia.

## Archivos necesarios

1. **cedeu-logo.png** - Logo de CEDEU
   - Tamaño recomendado: 300x300px
   - Formato: PNG con fondo transparente
   - Color fallback: #003366 (azul oscuro)

2. **santander-logo.png** - Logo de Santander Financial Innovation (BME)
   - Tamaño recomendado: 300x300px
   - Formato: PNG con fondo transparente
   - Color fallback: #0066CC (azul Santander)

3. **medusa-logo.png** - Logo de Medusa Capital
   - Tamaño recomendado: 300x300px
   - Formato: PNG con fondo transparente
   - Color fallback: #8B5CF6 (púrpura crypto)

## Nota

Si los logos no están disponibles, las tarjetas mostrarán el color fallback correspondiente, que se ve profesional y elegante.

## Optimización

Para optimizar las imágenes:
- Tamaño máximo: 100KB por imagen
- Resolución: 300x300px o similar
- Herramientas: TinyPNG, ImageOptim, o similar
